# Product Description

## Purpose

AI-eng financial dashboard is a training project to practice AI-assisted engineering workflows on a real full-stack codebase.

## What the product does

- Exposes financial movement data from a FastAPI backend.
- Aggregates and visualizes income, outcome, and profit trends in a React dashboard.
- Supports filtering and analytical endpoints (summary, top categories, alerts, period comparison, B2B/B2C slices).

## User value

- Helps learners reason about API contracts, data transformations, and KPI visualization.
- Provides a safe environment for creating and validating engineering rules with an AI agent.

## Current scope

- Demo and learning context.
- Uses seeded mock data, not persistent production data.
